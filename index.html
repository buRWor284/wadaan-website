ABOUT

WHO
WE WERE

Wadaan – The Web Development Network was a professional web design and development organisation based in Västerås, Sweden, providing web solutions to businesses all over the world.

Whatever clients were looking for — redesigning an existing website, from-scratch web application development, SEO, eCommerce, hosting or domain registration — Wadaan was there to help.

"Equipped with creative minds, fast fingertips, and the latest technology, Wadaan takes great pride in diverse and exciting designs and dynamic web applications."
