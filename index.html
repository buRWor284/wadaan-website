Who We Were

About Wadaan

Wadaan – The Web Development Network was a professional web design and development organisation based in Västerås, Sweden, providing web solutions to businesses all over the world. We created and managed websites that were not only aesthetically pleasing but also functionally superior.

Whatever clients were looking for — redesigning an existing website, from-scratch web application development, SEO, eCommerce, hosting or domain registration — Wadaan was there to help. For two years we built a reputation for creating a positive return on investment for our clients.

"Equipped with creative minds, fast fingertips, and the latest technology, Wadaan takes great pride in diverse and exciting designs and dynamic web applications. From simple text & graphics to more intricate designs, logos, and database-driven websites, Wadaan provides amazing services at very affordable prices." — Welcome post, April 12, 2004 (founding day)

Friday, 01 April 2005

E-Commerce Website 'Florida Tickets' is Done

This website was quite a simple one. We were assigned the task of applying a template to the vanilla version of osCommerce 2 MS 2.2. The client didn't have any PSD or HTML files. Our designer used the image supplied by the client and CSS to make changes. Buttons were also designed by Wadaan. Some other small changes were also made to the site.

www.floridatickets.org

2005 — News

IBM Pledges Patents to Open Source

IBM said it is pledging 500 software patents to developers of open-source software so they can develop without fear. The patents are available to those who abide by the Open Source Initiative (OSI) and use one of 50 OSI-certified licenses.

2006 — Portfolio Update

Leather Villa New York

One of our best works so far. Wadaan's job was the full design and backend provision and configuration for this site. osCommerce was used to set up the backend e-commerce script. The website has numerous great modules installed — a good admin authentication system, a Rich Text (WYSIWYG) Editor, and payment/shipping gateways including PayPal, Credit Card, and UPS.
