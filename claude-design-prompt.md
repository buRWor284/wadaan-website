# Claude Design Prompt — Wadaan.net Full Redesign

## What this site is

Redesign a single-page archive/memorial website for **Wadaan – The Web Development Network**, a web design and development company that was active from **April 2004 to 2007**, based in **Västerås, Sweden**. The site content was recovered from the Internet Archive Wayback Machine and is being preserved as a nostalgic digital artifact.

This is not a live business — it's an archive. The design should honor that era authentically.

---

## Design Direction: Retro-Nostalgic (2004–2007 Web Aesthetic)

Lean hard into **early-2000s web design aesthetics**, done tastefully and with craft — not mockingly. Think Windows XP era, the height of table-based layouts, corporate PHP developer sites, forum culture, and osCommerce storefronts. The result should feel like you opened a Wayback Machine snapshot but with the polish of someone who clearly cared about it.

### Visual references to draw from:
- **Windows XP / IE6 era UI patterns**: beveled/embossed button styles, inset borders, gradient title bars
- **Early corporate web blue** — deep navy (#003399 range) with gray silver accents
- **Pixel / bitmap fonts** for headings (VT323, Press Start 2P, or similar retro web fonts from Google Fonts)
- **Monospace courier-style** for code references, URLs, timestamps
- **Subtle tiled or patterned backgrounds** — think light crosshatch, subtle noise, or a very faint grid — not flat white
- **Status bars** at the bottom of sections (like old browsers)
- **Windows-style bordered cards/panels** — with a title bar at the top in a gradient blue, white body, and a thin outset or raised border
- **Table-style grid data** — the portfolio section should feel like a data table from an early web app, not modern pill tags
- **Scroll-triggered "loading" gimmick** (just a cute CSS detail referencing the old progress bar aesthetic)
- **Bold horizontal rules / dividers** with a 3D inset look (`border: 2px inset`)
- **Star / bullet GIF substitutes** — use ✦ ▸ or ◈ as decorative list bullets
- **Footer** styled like an old browser status bar or taskbar, dark gray, monospace text

### Color palette:
- Primary: `#003580` or `#1a4472` (classic early-web navy)
- Accent: `#c0392b` or `#cc3300` (links and dates in warm red)
- Chrome: `#d4d0c8` (Windows XP silver/gray) for borders and panel backgrounds
- Body background: very light warm gray or subtle texture — NOT pure white
- Text: near-black `#1a1a1a`
- Panel/card title bars: gradient from `#1a4472` to `#3a6498`

### Typography:
- Headings: **VT323** or **Press Start 2P** (Google Fonts) at larger sizes, or alternatively **Orbitron** if the pixel fonts feel too heavy
- Body: Georgia or a serif for the editorial content
- UI/labels/nav: a clean sans (Verdana, Tahoma, or Arial — all period-appropriate)
- Code/URLs: Courier New or Lucida Console

### Layout feel:
- Max width ~960px (period-appropriate), centered, with a subtle drop shadow on the outer container
- The "chrome" around the site should feel like a desktop app window
- Navigation should feel like an old-school horizontal tab bar or a Windows menu bar
- Section headers should use the Windows-style title bar aesthetic (gradient blue bar, white text, maybe a small icon)

## Additional Design Details

- Add a **fake "Site Counter"** widget (like the hit counters of that era) in the sidebar or footer — style it as an odometer display but show "visitors: 0004,821" just for flavor
- The **cursor** on links should feel slightly retro — maybe a pointer but make the hover states very visible (underline + color shift, like old browsers defaulted to)
- **Scrollbar styling** (where supported): style the scrollbar in old IE fashion — navy track, silver thumb
- Keep the site as a **single HTML file** — period-appropriate
- Every section header should feel like a **Windows application panel** with a colored gradient title bar and raised/sunken border
- Do NOT add animations that feel modern (no fade-ins, no parallax). Subtle blink or marquee CSS is acceptable if used sparingly and in a clearly tongue-in-cheek way
- Add a small **"Best viewed in Internet Explorer 6 at 800×600"** footnote somewhere — a classic era touch

---

## Output

Single `index.html` file. All CSS inline in `<style>` tags. No external frameworks. Use Google Fonts CDN for the retro typeface only. Keep it faithful to the era: semantic but table-influenced layouts, functional, dense with information.
